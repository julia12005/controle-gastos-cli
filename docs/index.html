Entrega Intermediária

Controle de Gastos CLI

Aplicação em Python para controle de gastos pessoais via terminal, evoluída com integração a uma API pública de cotação de moedas.

Ver Repositório Como Executar

Sobre o Projeto

O projeto ajuda usuários a registrar despesas, listar gastos e consultar o total gasto de forma simples, usando uma interface de linha de comando.

Funcionalidades

  • Adicionar gastos com descrição e valor
  • Listar gastos registrados
  • Consultar total de despesas
  • Consultar cotação atual do dólar
  • Converter valores em dólar para reais

API Pública

A aplicação consome a AwesomeAPI para consultar a cotação USD-BRL em tempo real.

https://economia.awesomeapi.com.br/json/last/USD-BRL

Fluxo de Desenvolvimento

A evolução foi feita com Issue no GitHub, branch entrega-intermediaria, commits versionados, Pull Request e merge para a branch principal.

Como Executar

git clone https://github.com/julia12005/controle-gastos-cli.git
cd controle-gastos-cli
pip install -r requirements.txt
python -m src.main
